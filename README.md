# GEMENI-chart-sentense-improver
AI chart generator + sentence improver built with Google Gemini and Streamlit.


 Gemini Chart + Sentence Improver

An AI-powered web application that answers your questions with intelligent chart generation and transforms your sentences into professional, polished text — all powered by Google Gemini.


Features

Chart Q&A
Ask any question and get a smart, data-driven answer
Automatically generates relevant charts and graphs based on your query
Visualizes information clearly for better understanding

entence Improver
Grammar Fix — Corrects grammatical errors in your text
Rephrase & Rewrite — Rewrites your sentences in a more professional and fluent tone
Supports both fixing and rephrasing in one go


Tech stack
Frontend / UI - Streamlit
Language - Python
AI Model - Google Gemini API(Flash 3.0)
